# Proyecto de Gestión de Universidad 🏫

Aplicación Full Stack para la administración y control de registros de docentes universitarios.

## 🛠️ Tecnologías Utilizadas
* **Frontend:** React.js, HTML5, CSS3 (Diseño responsivo/monocromático).
* **Backend:** Node.js, Express.js.
* **Base de Datos:** MySQL.
* **Seguridad:** Variables de entorno con `dotenv`.

## 🚀 Instrucciones de Instalación
1. Clona este repositorio.
2. Crea un archivo `.env` en la carpeta `server` usando como base el `.env.example`.
3. Ejecuta `npm install` tanto en la carpeta `/client` como en `/server`.
4. Inicia los servidores de desarrollo con `npm run dev`.
