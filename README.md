# Proyecto de Gestión de Universidad 🏫

Aplicación Full Stack (MERN/PERN-stack adaptada) diseñada para la administración, control y registro de docentes universitarios. Desarrollado como proyecto práctico para la asignatura de Frameworks Full Stack.

---

## 🛠️ Tecnologías y Arquitectura

El proyecto está estructurado de forma modular separando el entorno del cliente y del servidor:

* **Frontend (Client):** Desarrollado en **React.js**, utilizando una interfaz limpia, responsiva y estructurada con CSS3.
* **Backend (Server):** Construido sobre **Node.js** utilizando el framework **Express.js** para la creación de una API REST segura.
* **Base de Datos:** **MySQL**, gestionada de forma relacional para garantizar la integridad de los datos de los docentes.
* **Seguridad:** Gestión de credenciales críticas mediante variables de entorno con la librería `dotenv`.

---

## 📂 Estructura del Repositorio

* `📁 client/` -> Código fuente del frontend (React).
* `📁 server/` -> Código fuente del backend (Node/Express) y configuración de rutas API.
* `📁 database/` -> Contiene el script estructurado para montar la base de datos local.
